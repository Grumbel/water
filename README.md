Water
=====

A simple OpenGL/Glut based water toy.

| key | function                   |
|:---:|----------------------------|
| ESC | exit                       |
| SPC | start/stop simulation      |
|  e  | sync both water arrays     |
|  o  | swap water arrays          |
|  a  | 3D shake                   |
|  1  | decrease wave displacement |
|  2  | increase wave displacement |
|  c  | reset water simulation     |
